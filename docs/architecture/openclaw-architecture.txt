架构说明 (Architecture Description):
1. 数据产生 (Source): OpenClaw Agent 在执行过程中产生 JSONL 日志及 OTel 遥测数据。
2. 数据采集 (Ingestion): Vector 实时监听本地文件，OTel Collector 采集指标与链路追踪。
3. 数据存储 (Storage): 经过清洗后的数据通过 Stream Load 高吞吐进入 Apache Doris 数据库。
4. 数据应用 (Insight): Backend 提供 API，Frontend 进行可视化展示与故障诊断。
